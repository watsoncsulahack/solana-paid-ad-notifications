Paid Agent Skill — Delivery Gateway

Use this as your monetized skill offering for paid ad delivery verification and trigger orchestration.

Monetization Model

  • Publish skill with a base lamport price.
  • Client asks quote for N units.
  • Client pays on-chain.
  • Gateway executes only after payment verification.

API Scaffold (web2.5 backend)

GET  /api/agent/health
POST /api/agent/skills
GET  /api/agent/skills
POST /api/agent/quote
POST /api/agent/execute
